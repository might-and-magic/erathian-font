Introduction

On the continent of Erathia (a.k.a. Antagarich) in 3DO and New World Computing's game Heroes of Might and Magic III or Might and Magic VII, people speak Erathian. Similar to Common Tongue in A Song of Ice and Fire/Game of Thrones, Erathian language is actually English, but has different shapes of alphabet letters (see Erathian Alphabet Letters List below).

To help Earthlings to learn Erathian, I made this Erathian font. It works with special machines that are called "computers" on Earth. Enjoy it.

Download

Download either of the two font files: TTF (TrueType) Format | OTF (OpenType) Format

Installation

You may choose ONE OF these methods to install Erathian font on your operating system:

  • Open erathian.ttf, click "Install" button
  • OR, right click erathian.ttf, choose "Install"
  • OR, copy erathian.ttf to your system fonts folder (e.g. for Windows: C:\Windows\Fonts; for Ubuntu or other Unix/Linux, see Ubuntu Wiki; for Mac OS X, see Apple Support)

Tools

  • Earth Language-Erathian Translator: If you are a user of a web forum, GitHub, WordPress blog, etc., go to Earth Language-Erathian Translator and fetch the (U)BBCode/HTML/Markdown code which allow you to insert Erathian letters as images into your forum, GitHub .md file page, blog, etc.
  • User Script (erathian.user.js): This is a Greasemonkey/Tampermonkey/Violentmonkey userscript (how to use) to translate ALL webpages in your browser into Erathian.
  • Erathian.css: If you are a webmaster/website administrator/developer, you may use this CSS file and self-host the font files to use Erathian font in your websites and web apps.
  • Font template: If you want to make your font, you can use my font template to create one using Adobe Illustrator and FontForge, just like what I did here.

Erathian Alphabet Letters List

Basic Latin ABCDEFGHIJKLM
Erathian (Uppercase) ABCDEFGHIJKLM
Erathian (Lowercase) abcdefghijklm
Basic Latin NOPQRSTUVWXYZ
Erathian (Uppercase) NOPQRSTUVWXYZ
Erathian (Lowercase) nopqrstuvwxyz

Example Latin letters with diacritic:

Latin letter with diacritic ÂÉÌÖÇ
Erathian (Uppercase) ÂÉÌÖÇ
Erathian (Lowercase) âéìöç

The font has full coverage of A-Z based latin letters (which exclude ß, µ, Þ, þ, ð), with and without diacritic, in cp1252 and cp1250 (i.e. almost all Western and Central European languages).

/fonts/Erathian.[otf|ttf|woff|woff2] font files contain ALL the glyphs in Alegreya Medium font, while /fonts/min/Erathian-min.[otf|ttf|woff|woff2] contain only the essential glyphs (cp1252 and cp1250).
